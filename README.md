# 🍽️ Zomato Restaurant Data Analysis

## 📌 Project Overview

This project was completed as part of my **Data Analyst Internship at SAMAI Technologies**.

The objective of this project is to analyze restaurant data, generate meaningful business insights, build visualizations, develop a machine learning model to predict restaurant ratings, and create a restaurant recommendation system using the Zomato Restaurants dataset.

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing
- Analyze restaurant distribution across different cities
- Study online delivery trends
- Analyze restaurant price ranges
- Build a Machine Learning model to predict restaurant ratings
- Develop a Restaurant Recommendation System
- Create an interactive Power BI Dashboard

# 📊 Project Tasks

## ✅ Task 1 – Data Cleaning & City-wise Restaurant Analysis

- Loaded the dataset
- Explored dataset structure
- Checked missing values
- Removed duplicate records
- Performed city-wise restaurant analysis
- Calculated average ratings by city
- Created a bar chart of the top 10 cities
- Saved the cleaned dataset

---

## ✅ Task 2 – Online Delivery Analysis

- Analyzed restaurants offering online delivery
- Calculated online delivery percentages
- Compared ratings of restaurants with and without online delivery
- Created pie and bar charts

---

## ✅ Task 3 – Price Range Analysis

- Analyzed restaurant price ranges
- Calculated average ratings for each price range
- Identified the highest-rated price category
- Created bar and pie charts

---

## ✅ Task 4 – Restaurant Rating Prediction

Built a Machine Learning model using:

- Random Forest Regressor

### Features Used

- City
- Cuisines
- Price Range
- Votes
- Online Delivery

### Model Evaluation

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## ✅ Task 5 – Restaurant Recommendation System

Developed a recommendation system that suggests restaurants based on:

- City
- Cuisine
- Budget
- Minimum Rating

The system displays restaurants that best match the user's preferences.
